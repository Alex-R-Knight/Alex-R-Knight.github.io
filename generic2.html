Game Project: Goat Emulator

Using C++, I have made a game utilising a physics system with complex collision detection, multiple forms of artificial intelligence, and pathfinding.

The physics system provides physics simulation to all objects in the game, featuring spatial acceleration structures to improve performance when using many physics objects.
Collisions may be processed between spheres, axis-aligned bounding boxes (AABBs), and capsules.

Characters are controlled using both state machines and behaviour trees, the first to control a character patrolling a path unless disrupted, in which case it will flee.
The behaviour tree directs a character to patrol and search for the player, investigating disturbances and pursuing the player if they are found.
Both of these utilise an A* pathfinding algorithm.

A menu state and pausing the game is achieved through the use of pushdown automata.

Made using C++, in Visual Studio 2019.
